# Usage

Vue i18n Dashboard can be used in two main ways.

---

## 🟢 Local Mode

### Concept

Run the dashboard inside a single project.

### Workflow

1. Start dashboard
2. Scan project
3. Edit translations
4. Export JSON

### Use cases

- Solo development
- Small teams
- Quick setup

---

## 🔵 Server Mode

### Concept

Run a centralized dashboard for multiple projects.

### Workflow

1. Add projects (Git or local)
2. Sync translations
3. Use API in apps
4. Manage translations centrally

### Benefits

- No JSON file management
- Centralized workflow
- Scalable system

---

## Recommendation

- Local Mode → simplicity
- Server Mode → scalability
